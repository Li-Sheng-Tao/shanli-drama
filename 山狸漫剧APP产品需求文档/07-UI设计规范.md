## 一、全局设计令牌 (Design Tokens)

### 1.1 颜色系统

```scss
// ============================================
// 品牌色
// ============================================
$color-primary:       #FF4757;    // 主色调 - 珊瑚红（用于强调、激活状态）
$color-primary-dark:  #E8384F;    // 主色深色（按下状态）
$color-accent-pink:   #EC4899;    // 粉色（社交互动：关注按钮）
$color-accent-gold:   #F5D547;    // 金黄色（VIP、成就、高亮）
$color-accent-orange: #FF6B35;    // 橙色（福利页CTA、任务按钮）
$color-accent-teal:   #00B8A9;    // 青色（独家标签）

// ============================================
// 中性色
// ============================================
$color-bg-primary:    #000000;    // 播放页背景（纯黑）
$color-bg-secondary:  #FFFFFF;    // 找片/我的页背景（纯白）
$color-bg-card:       #FFFFFF;    // 卡片背景
$color-bg-gray:       #F5F5F5;    // 灰色背景（福利页内容区）
$color-bg-input:      #F2F2F7;    // 输入框/搜索框背景

// ============================================
// 文字色
// ============================================
$color-text-primary:    #FFFFFF;  // 主文字（播放页）
$color-text-dark:       #1A1A1A;  // 主文字（浅色页面）
$color-text-secondary:  #9CA3AF;  // 副文字（灰色）
$color-text-muted:      #6B7280;  // 弱文字（导航非激活态）
$color-text-tag:        #8E8E93;  // 标签文字

// ============================================
// 功能色
// ============================================
$color-success: #52C41A;
$color-warning: #FAAD14;
$color-danger:  #FF4D4F;
$color-info:    #1890FF;

// ============================================
// 渐变
// ============================================
$gradient-welfare: linear-gradient(135deg, #FF4D4F, #FF7A45);       // 福利页头部
$gradient-ranking: linear-gradient(135deg, #8B5CF6, #EC4899);       // 排行榜头部
$gradient-vip-dark: linear-gradient(135deg, #1A1A2E, #16213E);      // VIP卡片背景
```

### 1.2 字体系统

```scss
// ============================================
// 字体族
// ============================================
$font-family-base: -apple-system, BlinkMacSystemFont, 'PingFang SC',
  'Helvetica Neue', 'Microsoft YaHei', sans-serif;

// ============================================
// 字号
// ============================================
$font-size-xs:    10px;   // 极小文字（角标）
$font-size-sm:    12px;   // 小字（标签、辅助信息）
$font-size-base:  14px;   // 正文
$font-size-md:    16px;   // 标题/导航
$font-size-lg:    18px;   // 页面标题
$font-size-xl:    20px;   // 大标题
$font-size-xxl:   24px;   // 数字展示

// ============================================
// 字重
// ============================================
$font-weight-normal:  400;
$font-weight-medium:  500;
$font-weight-bold:    700;
```

### 1.3 间距系统

```scss
// 基于 4px 基数
$spacing-xs:  4px;
$spacing-sm:  8px;
$spacing-md:  12px;
$spacing-base: 16px;
$spacing-lg:  20px;
$spacing-xl:  24px;
$spacing-xxl: 32px;

// 页面边距
$margin-page: 16px;

// 卡片间距
$gap-card: 12px;
```

### 1.4 圆角

```scss
$radius-sm:    4px;    // 小元素
$radius-base:  8px;    // 卡片、按钮
$radius-md:    12px;   // 大卡片
$radius-lg:    16px;   // 弹窗
$radius-pill:  9999px; // 胶囊形（标签、按钮）
$radius-circle: 50%;   // 圆形（头像）
```

### 1.5 阴影

```scss
$shadow-card:    0 2px 8px rgba(0, 0, 0, 0.06);
$shadow-card-up: 0 4px 16px rgba(0, 0, 0, 0.1);
$shadow-float:   0 8px 24px rgba(0, 0, 0, 0.15);
```

### 1.6 Z-Index 层级

```scss
$z-base:      1;
$z-card:      10;
$z-sticky:    100;
$z-navbar:    200;
$z-dropdown:  300;
$z-modal-bg:  400;
$z-modal:     500;
$z-toast:     600;
$z-tooltip:   700;
```
