# 山狸漫剧APP 产品概述文档

## 1. 产品基本信息

| 项目 | 内容 |
|------|------|
| 产品名称 | 山狸漫剧（正式名可能为《山狸看剧》） |
| 产品类型 | 短剧/漫剧视频播放应用 |
| 目标平台 | H5（移动端Web）→ 微信小程序 → APP（iOS/Android） |
| 开发优先级 | H5 优先开发验证效果，小程序同步适配，APP 后续原生开发 |
| 技术方案 | 前端采用 uni-app 跨端框架，后端采用 Python (FastAPI) |

---

## 2. 产品定位

### 2.1 核心定位
一款以看剧为核心功能的短剧视频应用，通过金币激励系统培养用户观看习惯，结合会员付费解锁和广告变现的商业模式。

### 2.2 核心价值主张
- **用户价值**：免费用户通过做任务领金币兑换解锁时长，降低付费门槛
- **商业价值**：广告变现 + 会员订阅双轨盈利模式
- **留存价值**：金币激励系统培养用户观看习惯，提升留存率

---

## 3. 目标用户

### 3.1 用户画像
| 用户类型 | 特征描述 | 核心需求 |
|---------|---------|---------|
| 免费用户 | 价格敏感，时间充裕 | 通过看广告/做任务免费看剧 |
| 付费会员 | 价格不敏感，追求体验 | 无广告、畅看所有剧集 |
| 创作者（二期） | 内容生产者 | 发布作品、获得收益 |

### 3.2 用户行为特征
- 碎片化观看习惯（早、中、晚不同时段）
- 对新剧、热剧有较高关注度
- 社交分享意愿较强
- 对福利活动敏感

## 4. 核心功能模块

### 4.1 功能架构图

```
山狸漫剧APP
├── 主屏模块
│   ├── A-刷刷（短视频流）
│   ├── B-找片（剧集发现）
│   ├── C-福利（金币任务）
│   └── D-我的（个人中心）
├── 播放模块
│   ├── 竖屏播放
│   ├── 横屏全屏播放
│   └── 剧集详情页
├── 积分系统
│   ├── 山狸币获取
│   ├── 山狸币兑换
│   └── 宝箱盲盒
├── 会员系统
│   ├── 会员权益
│   └── 会员购买
└── 后台管理
    ├── 内容管理
    ├── 用户管理
    └── 数据分析
```

### 4.2 功能模块概述

| 模块 | 功能描述 | 优先级 |
|------|---------|--------|
| 刷刷（A屏） | 短视频流播放，上下滑动切换剧集 | P0 |
| 找片（B屏） | 剧集搜索、筛选、排行榜、上新日历 | P0 |
| 福利（C屏） | 金币任务、宝箱兑换、会员时长兑换 | P0 |
| 我的（D屏） | 个人信息、会员状态、收藏记录 | P0 |
| 播放器 | 竖屏/横屏播放、倍速、弹幕评论 | P0 |
| 积分系统 | 金币获取与消耗逻辑 | P0 |
| 会员系统 | 会员购买与权益管理 | P0 |
| 广告系统 | 开屏广告、解锁广告 | P0 |
| 创作者平台 | 内容上传、收益分成（二期） | P1 |
| 营销功能 | 商城、直播、小游戏（二期） | P1 |

## 5. 商业模式

### 5.1 收入来源
1. **会员订阅**：付费会员解锁畅看 + 去广告
2. **广告收入**：开屏广告、解锁跳集广告
3. **增值服务**（二期）：创作者平台手续费、直播打赏分成

### 5.2 用户分层运营
| 用户层级 | 运营策略 |
|---------|---------|
| 免费用户 | 看广告解锁，金币任务激励留存 |
| 付费会员 | 畅看无广告，专属客服，小山狸视频福利 |
| 高价值用户 | 精准推送，专属活动 |

## 7. 开发阶段规划

### 7.1 第一阶段：H5 端（优先验证）
- H5 移动端 Web 全部核心功能
- 四大主屏功能
- 播放器核心功能（HLS/FLV 流媒体）
- 积分系统
- 会员系统
- 广告系统
- 后台管理系统
- **目标**：快速上线验证产品效果和用户反馈

### 7.2 第二阶段：小程序端
- 基于 uni-app 编译为微信小程序
- 小程序专属适配（登录授权、支付、分享等）
- 小程序审核与发布
- **目标**：借助微信生态获客

### 7.3 第三阶段：APP 端
- 基于 uni-app 编译为原生 APP（或后续原生开发）
- APP 专属功能（推送通知、桌面组件等）
- 应用商店上架
- **目标**：深度用户体验和留存

### 7.4 第四阶段：扩展功能（二期）
- 创作者平台
- 营销功能（商城、直播、小游戏）
- 课程业务绑定
- 合作业务
