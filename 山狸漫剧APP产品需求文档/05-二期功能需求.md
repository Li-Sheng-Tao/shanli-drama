# 山狸漫剧APP 二期功能需求文档

## 1. 二期功能概述

### 1.1 开发优先级
二期功能在一期核心功能完成后开发，包括：
- 创作者平台（PC端）
- 营销功能（商城、直播、小游戏）

### 1.2 功能架构

```
二期功能
├── 创作者平台（PC端）
│   ├── 账号入驻
│   ├── 作品上传
│   ├── 收益分成
│   └── 合作大厅
└── 营销功能
    ├── 商城页面
    ├── 直播功能
    └── 小游戏
```

---

## 2. 创作者平台

### 2.1 平台概述

| 项目 | 说明 |
|------|------|
| 平台名称 | 山狸创作者平台 |
| 平台类型 | PC端Web平台 |
| 参考产品 | 抖音创作者平台 |
| 核心功能 | 注册入驻、发漫剧、获得平台收益分成、平台推流补贴支持 |

### 2.2 用户角色

| 角色 | 说明 | 入驻方式 |
|------|------|---------|
| 普通创作者 | 发布个人作品 | 免费入驻 |
| 编剧课学员 | 发布剧本/小说 | 免费入驻（推荐码） |
| 荣誉会员 | 发布剧本/小说 | 课程赠送推荐码 |
| 承制方/客户 | 挑选作品合作 | 按月付费 |
| 作者方 | 发布作品 | 一次性入会费，终身免费 |

### 2.3 创作者入驻流程

#### 2.3.1 入驻步骤

```
1. 申请入驻
   ├── 填写基本信息
   ├── 提交资质材料
   └── 选择入驻类型

2. 平台审核
   ├── 资质审核
   └── 内容审核

3. 签约
   ├── 签署合作协议
   └── 确认分成比例

4. 开通账号
   └── 获得创作者权限

5. 上传作品
   ├── 上传视频/剧本
   ├── 填写作品信息
   └── 提交审核

6. 版权合规
   └── 平台帮助合规版权
```

#### 2.4.3 推流补贴

| 功能 | 说明 |

### 2.5 合作大厅

#### 2.5.1 功能概述

| 项目 | 说明 |
|------|------|
| 定位 | 稿件交易平台 |
| 形式 | 以稿件为主 |
| 参与方 | 付费会员、承制方、客户、作者 |

#### 2.5.2 交易流程

```
1. 发布需求/作品
   ├── 作者发布作品（剧本、小说）
   └── 客户发布需求

2. 浏览筛选
   ├── 承制方/客户浏览作品
   └── 作者浏览需求

3. 发起合作
   └── 发起合作意向

4. 达成合作
   ├── 确认合作内容
   └── 签署合作协议

5. 平台服务
   ├── 收取平台手续费
   └── 提供版权服务
```

#### 2.5.3 收费模式

| 角色 | 收费方式 |
|------|---------|
| 编剧课学员 | 免费入驻（推荐码） |
| 荣誉会员 | 课程赠送推荐码 |
| 承制方/客户 | 按月付费 |
| 作者方 | 一次性入会费，终身免费 |
| 合作达成 | 平台手续费 + 版权服务费 |

| 元素 | 说明 |
