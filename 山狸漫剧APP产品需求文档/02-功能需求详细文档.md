# 山狸漫剧APP 功能需求详细文档

## 1. 主屏模块概览

APP底部导航栏包含4个主Tab：

| Tab名称 | 图标 | 功能定位 |
|--------|------|---------|
| 刷刷 | - | 短视频流播放（首页） |
| 找片 | - | 剧集发现与搜索 |
| 福利 | - | 金币任务与兑换 |
| 我的 | - | 个人中心 |

---

## 2. A屏 - 刷刷（短视频流）

### 2.1 功能概述
类似抖音的短视频流播放体验，用户可上下滑动切换剧集，竖屏/横屏播放。

### 2.2 页面元素

#### 2.2.1 视频播放区域
| 元素 | 说明 |
|------|------|
| 视频画面 | 支持竖屏和横向全屏播放 |
| 播放控制 | 空点画面暂停/播放 |
| 点赞交互 | 双击画面为点赞 |

#### 2.2.2 右侧交互栏（从上到下）
| 元素 | 位置 | 功能说明 |
|------|------|---------|
| 观看时长提示 | 头像上方 | 显示今日观看时长，优先提示"今日观看领金币任务"，完成后提示其他金币任务 |
| 发布者头像 | 右侧 | 可点击关注；前期显示官方主页，后期接入创作者主页 |
| 点赞按钮 | 右侧 | 点赞/取消点赞 |
| 收藏按钮 | 右侧 | 收藏到个人收藏夹 |
| 评论按钮 | 右侧 | 打开评论面板 |
| 分享按钮 | 右侧 | 分享到社交平台 |

#### 2.2.3 底部信息区域
| 元素 | 说明 |
|------|------|
| 剧名 | 点击可跳转剧集介绍页 |
| 热度与分类 | 显示剧集热度值和分类标签 |
| 集数与单集介绍 | 当前播放集数和该集简介 |
| 集数显示 | 点击可切换剧集首页/切换集数 |

### 2.3 交互逻辑

#### 2.3.1 滑动交互
| 操作 | 行为 |
|------|------|
| 上滑 | 切换到下一部剧（新剧） |
| 下滑 | 返回上一部剧 |
| 播放完当前剧集 | 自动跳转当前剧集的第二集 |
| 看完第二集 | 提示收藏/弹出付费引导 |

#### 2.3.2 播放器功能
| 功能 | 说明 |
|------|------|
| 倍速播放 | 支持0.5x/0.75x/1x/1.25x/1.5x/2x |
| 弹幕开关 | 开启/关闭弹幕显示 |
| 全屏切换 | 竖屏↔横屏全屏 |

### 2.4 评论功能

#### 2.4.1 评论面板
| 功能 | 说明 |
|------|------|
| 评论列表 | 显示所有评论，支持按热度/时间排序 |
| 发表评论 | 输入框 + 发送按钮 |
| 表情包 | 支持插入表情包 |
| 弹幕模式 | 评论可发送为弹幕 |

#### 2.4.2 评论交互
| 操作 | 行为 |
|------|------|
| 点赞评论 | 点赞/取消点赞 |
| 回复评论 | 回复某条评论 |
| 举报评论 | 举报不当内容 |

### 2.5 剧集首页（点击剧名进入）

#### 2.5.1 页面结构
| 区域 | 内容 |
|------|------|
| 顶部 | 剧集封面、剧名、热度、分类 |
| 中部 | 剧集简介、演员信息 |
| 底部 | 剧集列表（横向滑动选择集数） |
| 操作栏 | 收藏、分享、开始观看按钮 |

## 3. B屏 - 找片（剧集发现）

### 3.1 B-01 主页面

#### 3.1.1 顶部导航
| 元素 | 功能 |
|------|------|
| 搜索框 | 输入关键词搜索剧集 |
| 筛选按钮 | 打开筛选面板 |
| 排行榜 | 跳转排行榜页面 |
| 创作者 | 跳转创作者列表（二期） |
| 上新 | 跳转上新日历页面 |

#### 3.1.2 剧集列表
| 元素 | 说明 |
|------|------|
| 排列方式 | 双排一屏4部剧 |
| 剧集封面 | 左上角推荐水印，右下角连载/完结标签 |
| 剧集信息 | 剧名、分类、热度 |

#### 3.1.3 广告弹窗规则
| 规则 | 说明 |
|------|------|
| 弹出频率 | 每4小时弹出一次广告页面 |
| 广告内容 | 推广广告/新剧热剧 |
| 金币任务 | 点击广告完成当日广告金币任务 |
| 每日上限 | 24小时内可领取6次 |

### 3.2 B-02 筛选页面

#### 3.2.1 筛选维度
| 维度 | 选项 |
|------|------|
| 题材 | 都市、古装、悬疑、甜宠等 |

| 看点 | 虐恋、爽文、复仇等 |
| 画风 | 真人、动漫、漫画等 |
| 排序 | 最新、最热、评分最高 |

#### 3.2.2 剧集卡片
| 元素 | 说明 |
|------|------|
| 封面 | 左上角标注独家/上榜/新剧，右下角标注连载/完结 |
| 剧名 | 显示剧名 |
| 分类标签 | 显示分类信息 |

### 3.3 B-03 排行榜页面

#### 3.3.1 榜单类型
| 榜单 | 排名依据 |
|------|---------|
| 热播榜 | 播放次数 |
| 热搜榜 | 搜索次数 |
| 收藏榜 | 收藏次数 |
| 预约榜 | 预约数量 |

#### 3.3.2 榜单展示
| 元素 | 说明 |
|------|------|
| 排名显示 | 123名高亮显示 |
| 数据显示 | 显示具体数据值 |
| 差距显示 | 显示与临近两名的差距 |
| 剧集信息 | 显示封面与剧名 |
| 交互 | 点击可跳转播放剧集 |

#### 3.3.3 排行说明
- 右上角弹出排行说明弹窗
- 说明各榜单计算规则

### 3.4 B-04 活动页面

#### 3.4.1 页面结构
| 元素 | 说明 |
|------|------|
| 返回按钮 | 左上角返回 |
| 广告屏切换 | 支持3屏广告页左右切换 |
| 活动内容 | 官方活动/合作广告页 |
| 跳转链接 | 点击可跳转外部链接 |

### 3.5 B-05 上新日历页面

#### 3.5.1 页面结构
| 元素 | 说明 |
|------|------|
| 日历组件 | 当日高亮显示，可切换日期 |
| 日期提示 | 今日日期 + 祝福语/推剧广告 |
| 分类标签 | 新剧/连载更新 |

#### 3.5.2 剧集卡片
| 元素 | 说明 |
|------|------|
| 剧集封面 | 显示剧集封面图 |
| 剧集名称 | 显示剧名 |
| 分类与看点 | 显示分类标签和看点 |
| 操作按钮 | 预约/收藏 |

#### 3.5.3 预约功能
| 功能 | 说明 |
|------|------|
| 新剧预约 | 新剧上架时提醒用户 |
| 更新提醒 | 剧集更新时提醒用户 |

#### 3.5.4 用户喜好收集
- 当日首次进入弹出用户喜好/评价收集
- 完成后可领取金币奖励

## 6. 广告系统

### 6.1 广告类型

| 广告类型 | 触发场景 | 说明 |
|---------|---------|------|
| 开屏广告 | 打开APP时弹出 | 全屏广告展示 |
| 解锁广告 | 解锁剧集跳集时 | 观看广告解锁下一集 |
| 弹窗广告 | 找片页面每4小时 | 推广广告/新剧热剧 |

### 6.2 广告规则

| 规则 | 说明 |
|------|------|
| 免费用户 | 需要看很多广告 |
| 会员用户 | 免广告权益 |
| 新剧保护期 | 看保护期新剧解锁需要消耗更多金币 |

### 6.3 广告金币任务
- 点击弹窗广告完成当日广告金币任务
- 24小时内可领取6次

## 8. 获客系统
