## 核心业务规则速查

### 山狸币规则
- **获取方式**：完成任务获得（不可充值购买）
- **用途**：兑换解锁时长、兑换宝箱盲盒
- **兑换规则**：X山狸币 = 解锁时长（不可去广告）
- **详细规则**：见 [03-积分与会员系统](./03-积分与会员系统.md)

### 会员权益
- **核心权益**：畅看所有剧集 + 去广告
- **专属福利**：VIP客服、小山狸视频、随机山狸币赠送
- **详细权益**：见 [03-积分与会员系统](./03-积分与会员系统.md)

### 剧集解锁规则
- **会员用户**：直接观看，无需解锁
- **免费用户**：金币解锁 / 看广告解锁
- **新剧保护期**：消耗更多金币
- **详细规则**：见 [02-功能需求详细文档](./02-功能需求详细文档.md)

### 广告规则
- **开屏广告**：打开APP时弹出
- **解锁广告**：解锁剧集跳集时
- **弹窗广告**：找片页面每4小时，每日6次
- **详细规则**：见 [02-功能需求详细文档](./02-功能需求详细文档.md)

## 开发注意事项

### 优先级说明
- **P0**：一期核心功能，必须完成
- **P1**：二期扩展功能，后续完成

### 数据埋点
所有关键业务操作需要埋点，详见各文档中的数据埋点需求章节。

### 安全要求
- 全站HTTPS加密
- JWT Token认证
- 视频DRM加密
- 内容审核机制
