# 山狸漫剧APP 积分与会员系统文档

## 1. 山狸币系统

### 1.1 山狸币定义

| 属性 | 说明 |
|------|------|
| 名称 | 山狸币 |
| 用途 | 兑换解锁时长、兑换宝箱 |
| 获取方式 | 完成任务获得（不可充值购买） |
| 核心目的 | 激励用户留存时长和看剧习惯培养 |

### 1.3 山狸币消耗规则

#### 1.3.1 兑换会员时长

| 兑换项 | 消耗金币 | 权益说明 |
|--------|---------|---------|
| 1小时解锁时长 | X山狸币 | 可解锁剧集，不可去广告 |
| 3小时解锁时长 | X山狸币 | 可解锁剧集，不可去广告 |
| 1天解锁时长 | X山狸币 | 可解锁剧集，不可去广告 |

**注意**：山狸币兑换的时长仅用于解锁剧集，不包含去广告权益。

#### 1.3.2 兑换宝箱盲盒

| 宝箱类型 | 消耗金币 | 可能奖励 |
|---------|---------|---------|
| 普通宝箱 | X山狸币 | 金币/优惠券 |
| 精美宝箱 | X山狸币 | 金币/周会员/优惠券 |
| 豪华宝箱 | X山狸币 | 金币/周会员/好礼/现金/优惠券 |

#### 1.3.3 宝箱奖励概率配置

| 奖励类型 | 普通宝箱 | 精美宝箱 | 豪华宝箱 |
|---------|---------|---------|---------|
| 金币 | 60% | 40% | 30% |
| 周会员 | - | 5% | 10% |
| 好礼/实物 | - | - | 5% |
| 现金 | - | 3% | 5% |
| 优惠券 | 40% | 52% | 50% |

### 1.4 剧集解锁金币消耗

#### 1.4.1 解锁规则

| 剧集类型 | 解锁金币消耗 | 说明 |
|---------|-------------|------|
| 普通剧集 | 基础金币数 | 标准消耗 |
| 热门剧集 | 基础金币数 × N倍 | 根据排名阶梯设置 |
| 新剧（保护期） | 基础金币数 × N倍 | 保护期内消耗更多 |
| 独家剧集 | 基础金币数 × N倍 | 独家内容加成 |

#### 1.4.2 金币系数控制

后台可配置以下系数：
- 基础解锁金币数
- 热剧加成系数
- 新剧保护期系数
- 独家内容加成系数

## 2. 会员系统

### 2.1 会员等级

| 等级 | 名称 | 开通方式 |
|------|------|---------|
| 普通用户 | 免费用户 | 注册即可 |

| 权益 | 免费用户 | VIP会员 |
|------|---------|---------|
| 观看剧集 | 看广告解锁/金币解锁 | 畅看所有剧集 |
| 广告体验 | 有广告 | 无广告 |
| 解锁限制 | 需消耗金币/看广告 | 无限制 |
| 客服支持 | 普通客服 | VIP专属客服 |
| 小山狸视频 | 无 | 每日看剧任务完成后随机获得 |
| 山狸币赠送 | 无 | 小山狸视频随机赠送 |

### 2.3 会员购买方案

| 方案 | 价格 | 时长 | 说明 |

| 月度会员 | ¥X | 30天 | 按月订阅 |
| 季度会员 | ¥X | 90天 | 按季订阅 |
| 年度会员 | ¥X | 365天 | 按年订阅 |

#### 2.3.2 自动续费规则

| 规则 | 说明 |
|------|------|

### 2.4 会员专属福利

## 3. 用户分层运营

### 3.1 用户分层标准

| 用户层级 | 判断标准 | 运营策略 |
|---------|---------|---------|
| 新用户 | 注册7天内 | 新人任务引导、首充优惠 |
| 活跃免费用户 | 日活、高观看时长 | 金币任务激励、会员转化 |
| 付费会员 | 已购买会员 | 专属福利、续费维护 |
| 流失用户 | N天未登录 | 召回活动、推送唤醒 |

### 3.2 差异化运营

#### 3.2.1 免费用户
- 看广告解锁剧集
- 金币任务激励留存
- 会员转化引导

#### 3.2.2 付费会员
- 畅看无广告
- 小山狸视频福利
- VIP专属客服
- 随机山狸币赠送

## 4. 数据埋点需求

### 4.1 山狸币数据

| 埋点事件 | 说明 |
|---------|------|
| coin_earn | 金币获得（任务类型、数量、来源） |
| coin_spend | 金币消耗（消耗类型、数量、目标） |
| coin_balance | 金币余额查询 |
| task_complete | 任务完成（任务类型、奖励） |

### 4.2 会员数据

| 埋点事件 | 说明 |
|---------|------|
| member_purchase | 会员购买（方案、价格、渠道） |
| member_renew | 会员续费 |
| member_expire | 会员到期 |
| member_cancel | 取消续费 |

### 4.3 宝箱数据

| 埋点事件 | 说明 |
|---------|------|
| box_open | 打开宝箱（类型、消耗金币） |
| box_reward | 宝箱奖励（类型、内容） |

## 5. 后台配置需求

| 配置项 | 说明 |
|--------|------|
