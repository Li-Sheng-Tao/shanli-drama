# 山狸漫剧APP 技术需求文档

## 1. 技术架构概述

### 1.1 系统架构

```
┌─────────────────────────────────────────────────────────────┐
│                        客户端层                               │
├─────────────────┬─────────────────┬─────────────────────────┤
│   iOS APP       │  Android APP    │      小程序              │
└─────────────────┴─────────────────┴─────────────────────────┘
                              │
                              ▼
┌─────────────────────────────────────────────────────────────┐
│                        API网关层                             │
│              (负载均衡、鉴权、限流)                           │
└─────────────────────────────────────────────────────────────┘
                              │
                              ▼
┌─────────────────────────────────────────────────────────────┐
│                        服务层                                │
├─────────────┬─────────────┬─────────────┬───────────────────┤
│  用户服务    │  内容服务    │  积分服务    │    会员服务        │
├─────────────┼─────────────┼─────────────┼───────────────────┤
│  播放服务    │  消息服务    │  广告服务    │    数据服务        │
└─────────────┴─────────────┴─────────────┴───────────────────┘
                              │
                              ▼
┌─────────────────────────────────────────────────────────────┐
│                        数据层                                │
├─────────────────┬─────────────────┬─────────────────────────┤
│     MySQL       │      Redis      │       OSS/CDN           │
│   (业务数据)     │    (缓存/队列)   │    (视频/图片存储)        │
└─────────────────┴─────────────────┴─────────────────────────┘
```

### 1.2 技术栈建议

| 层级 | 技术选型 |
|------|---------|
| 客户端 | iOS(Swift)、Android(Kotlin)、小程序(原生/Taro) |
| API网关 | Nginx/Kong |
| 后端服务 | Node.js/Java/Go |
| 数据库 | MySQL 8.0+ |
| 缓存 | Redis 6.0+ |
| 消息队列 | RabbitMQ/Kafka |
| 对象存储 | 阿里云OSS/腾讯云COS |
| CDN | 阿里云CDN/腾讯云CDN |
| 搜索引擎 | Elasticsearch |
| 日志系统 | ELK Stack |

## 4. 核心业务流程

### 4.1 剧集解锁流程

```
┌─────────────┐
│ 用户请求解锁 │
└──────┬──────┘
       │
       ▼
┌─────────────────┐
│ 判断用户是否会员 │
└──────┬──────────┘
       │
   ┌───┴───┐
   │       │
   ▼       ▼
┌─────┐ ┌─────┐
│ 是  │ │ 否  │
└──┬──┘ └──┬──┘
   │       │
   ▼       ▼
┌─────┐ ┌─────────────┐
│直接  │ │判断解锁方式  │
│播放  │ └──────┬──────┘
       │        │
       │   ┌────┴────┐
       │   │         │
       │   ▼         ▼
       │ ┌─────┐ ┌─────┐
       │ │金币 │ │广告 │
       │ └──┬──┘ └──┬──┘
       │    │       │
       │    ▼       ▼
       │ ┌─────┐ ┌─────┐
       │ │扣除 │ │播放 │
       │ │金币 │ │广告 │
       │ └──┬──┘ └──┬──┘
       │    │       │
       └────┴───┬───┘
                 │
                 ▼
          ┌─────────────┐
          │ 记录解锁状态 │
          └──────┬──────┘
                 │
                 ▼
          ┌─────────────┐
          │  开始播放   │
          └─────────────┘
```

```
       │
   ┌───┴───┐
   │       │
   ▼       ▼
┌─────┐ ┌─────┐
│完成 │ │未完成│

   │
   ▼
┌─────────────┐
│ 用户领取奖励 │

       │
       ▼
┌─────────────┐

└──────┬──────┘
       │
       ▼
┌─────────────┐
│ 记录流水    │
└─────────────┘
```

|---------|------|
| 访问日志 | 记录所有API访问 |
| 业务日志 | 记录关键业务操作 |
| 错误日志 | 记录系统错误 |
| 审计日志 | 记录敏感操作 |
