# 山狸漫剧APP 后台管理系统需求文档

## 1. 后台系统概述

### 1.1 系统定位
为运营人员和管理员提供内容管理、用户管理、数据分析、活动管理等功能的Web后台管理系统。

### 1.2 用户角色

| 角色 | 权限范围 |
|------|---------|
| 超级管理员 | 所有功能 |
| 内容管理员 | 剧集管理、内容审核 |
| 运营管理员 | 活动管理、用户运营 |
| 数据分析师 | 数据查看、报表导出 |
| 客服人员 | 用户反馈处理 |

---

## 2. 内容管理模块

### 2.1 剧集管理

#### 2.1.1 剧集列表

| 字段 | 说明 |
|------|------|
| 剧集ID | 唯一标识 |
| 剧名 | 剧集名称 |
| 分类 | 题材、设定、看点、画风 |
| 状态 | 上架/下架/待审核 |
| 集数 | 总集数 |
| 热度 | 播放量/收藏量 |
| 上架时间 | 上架日期 |
| 标签 | 独家/新剧/热剧 |

#### 2.1.2 剧集操作

| 操作 | 说明 |
|------|------|
| 新增剧集 | 创建新剧集 |
| 编辑剧集 | 修改剧集信息 |
| 上架/下架 | 控制剧集可见性 |
| 删除剧集 | 删除剧集（软删除） |
| 批量操作 | 批量上架/下架/删除 |

#### 2.1.3 剧集信息编辑

| 字段 | 类型 | 必填 | 说明 |
|------|------|------|------|
| 剧名 | 文本 | 是 | 剧集名称 |
| 封面图 | 图片 | 是 | 剧集封面 |
| 简介 | 富文本 | 是 | 剧集介绍 |
| 分类标签 | 多选 | 是 | 题材、设定、看点、画风 |
| 演员信息 | 文本 | 否 | 主演信息 |
| 标签 | 多选 | 否 | 独家/推荐/新剧 |
| 解锁金币数 | 数字 | 是 | 解锁消耗金币 |
| 保护期 | 日期 | 否 | 新剧保护期时间 |

### 2.2 剧集批量上传

#### 2.2.1 批量上传功能

| 功能 | 说明 |
|------|------|
| 批量上传视频 | 支持批量上传剧集视频文件 |
| 批量上传封面 | 支持批量上传剧集封面图 |
| 批量导入信息 | 支持Excel导入剧集信息 |
| 单集介绍批量上传 | 支持批量上传每集的介绍文字 |

#### 2.2.2 上传流程

```
1. 准备上传文件（视频/图片/Excel）
2. 选择目标剧集或创建新剧集
3. 上传文件
4. 系统自动匹配/人工确认
5. 提交审核
6. 审核通过后上架
```

### 2.3 排行榜管理

#### 2.3.1 榜单配置

| 榜单类型 | 配置项 |
|---------|--------|
| 热播榜 | 排名依据（播放次数）、更新频率 |
| 热搜榜 | 排名依据（搜索次数）、更新频率 |
| 收藏榜 | 排名依据（收藏次数）、更新频率 |
| 预约榜 | 排名依据（预约数量）、更新频率 |

#### 2.3.2 榜单操作

| 操作 | 说明 |
|------|------|
| 手动调整 | 手动调整榜单排名 |
| 置顶推荐 | 将某剧集置顶显示 |
| 隐藏剧集 | 从榜单中隐藏某剧集 |

### 2.4 上新日历管理

#### 2.4.1 日历配置

| 功能 | 说明 |
|------|------|
| 新剧上架日期 | 设置新剧上架日期 |
| 更新计划 | 设置连载剧集更新计划 |
| 预约开关 | 开启/关闭剧集预约功能 |
| 提醒推送 | 设置上架提醒推送 |
